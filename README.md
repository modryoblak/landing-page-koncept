## Contributions ⭐

Ak chete niečo pridať, vytvorte si PR so zmenami. Nepovoľujeme pushe do main branche. 

### Commits

Commits v angličtine a v neurčitom čase, tagy vo forme `[Tag]` povolené.

Príklady:
```
Add navbar
```
```
[Npm] Remove unnecessary dependencies
```
